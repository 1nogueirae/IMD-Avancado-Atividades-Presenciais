# IMD Avancado - Atividades Presenciais

Repositorio para organizar e versionar as atividades presenciais do modulo avancado do curso de Informatica para Internet, no IMD/UFRN.

## Objetivo

Centralizar exercicios, anotacoes e materiais de aula em um unico lugar, com historico de evolucao por semana.

## Padrao de organizacao

- Cada semana deve ficar em uma pasta no formato `Semana-XX`.
- Arquivos de codigo devem ter nomes descritivos (ex.: `atividade2.js`).
- Materiais de apoio podem ficar em `.md`, `.pdf` e outros formatos necessarios.
